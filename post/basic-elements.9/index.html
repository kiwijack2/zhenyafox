успешно прошедшего производственную практику ПП.05 Производственная практика (по профилю специальности) в объеме 72 часов с 17.04.17 г. по 29.04.17г.

Место прохождения производственной практики

ООО «Информационно-технологический ресурс»

(наименование предприятия)

Виды и качество выполнения работ

Профессиональные компетенции Виды работ, выполненные студентом во время практики Качество выполненных работ в соответствии с технологией и (или) требованиями организации
ПК 5.1. Осуществлять ревьюирование кода и технической документации. Тестирование и отладка кода программного продукта в соответсвии с ГОСТ Р 8.596-2002. ГСИ. Соответствует выбранной технологии
ПК 5.2. Выполнять процесс измерения характеристик компонент программного продукта. Измерение характеристик компонент программного продукта в соответствии с МИ 2174-91 ГСИ. Соответствует выбранной технологии
ПК 5.3. Производить исследование и оптимизацию созданного программного кода с использованием специализированных программных средств. Исследование программного кода с использованием лицензионного программного обеспечения: «Visual Studio Team Suite» и оптимизация созданного программного кода в Visual Studio Team Suite. Соответствует выбранной технологии
ПК 5.4. Оказывать консультационную поддержку другим разработчикам в части реализации спроектированных компонент. Осуществление разработки программного проекта в MS Project, проведение анализа средств, затрат и сроков на программный проект. Использование метрик сложности для оценки качества программного проекта. Оказание консультационной поддержки разработчикам программных проектов исходя из выявленных погрешностей. Соответствует выбранной технологии

Производственная практика ПП.05 завершает изучение профессионального модуля ПМ. 05 «Участие в ревьюирование программных продуктов»

ХАРАКТЕРИСТИКА

Студент проходил производственную практику на

ООО «Информационно-технологический ресурс»

(наименование предприятия)

За время прохождения производственной практики с 17.04.17 г. по 29.04.17г. студент проявил себя следующим образом:

Отношение к работе: Ответственный

Соответствие выполненных работ требованиям квалификационной характеристике: Соответствует Трудовая дисциплина: Дисциплинирован

Соответствие профессиональных компетенций подготовке квалифицированных рабочих, служащих: Соответствует

В результате освоения данной программы ПП.05 Производственная практика (по профилю специальности) студент получил практический опыт по выполнению типовых операций, инсталляции, настройки, обслуживания и модификации программного обеспечения компьютерных систем; проявлял устойчивый интерес к своей будущей профессии.

Требования к выполнению работы Профессиональные компетенции Оценка
Тестирование и отладка кода программного продукта в соответсвии с ГОСТ Р 8.596-2002. ГСИ. ПК 5.1. Осуществлять ревьюирование кода и технической документации.
Измерение характеристик компонент программного продукта в соответствии с МИ 2174-91 ГСИ. ПК 5.2. Выполнять процесс измерения характеристик компонент программного продукта.
Исследование программного кода с использованием лицензионного программного обеспечения: «Visual Studio Team Suite» и оптимизация созданного программного кода в Visual Studio Team Suite. ПК 5.3. Производить исследование и оптимизацию созданного программного кода с использованием специализированных программных средств.
Осуществление разработки программного проекта в MS Project, проведение анализа средств, затрат и сроков на программный проект. Использование метрик сложности для оценки качества программного проекта. Оказание консультационной поддержки разработчикам программных проектов исходя из выявленных погрешностей. ПК 5.4. Оказывать консультационную поддержку другим разработчикам в части реализации спроектированных компонент.

Овладел общими компетенциями, включающими в себя способность: работать в команде; осуществлять поиск информации, необходимой для эффективного выполнения профессиональных задач; осуществлять текущий и итоговый контроль, делая оценку и коррекцию собственной деятельности.
